F.A.Q

  • Qual seria a vantagem do launcher?

    No launcher você tera mais segurança conta cheats ou injetores.
    Porque o launcher tem que esta sempre ativo no computador do cliente.
    Quando o cliente desligar ele ou da CTRL+F4 o cliente e deslogado do servidor.
    Com isto mantemos a maxima segurança no servidor.

  • O launcher tem suporte para VRP ou outros sistemas?

    Sim nosso launcher da suporte para qualquer sistema.

  • Como posso ajuda-lo?

    Voce pode ajuda agente fazendo uma doação para agente pelos sistemas.
    Paypal: Clique aqui para doar

  • Posso modificar o launcher?

    Podera sim.
    Desde que mantenha os creditos e links do ESXBrasil.

  • Como posso ter o launcher pro?

    Entre em nosso discord e pergunte la.
    Um dos nossos administradores falara com voce!

  • Este launcher tem o sistema anit-cheat?

    Não.
    O sistema anti-cheat e somente para os doadores do projeto.
    Faça uma doação em entre em contato com nossos administradores pelo discord.
    Para pegar o launcher com sistema anti-cheat.

  • Posso passa minha source para amigos?

    Não.
    Porque cada servidor tem suas configurações e o launcher ser voce for doador.
    Podera perde o sistema anti-cheat e suas atualizações.
